A very basic digital picture frame interface.
3D printed case courtesy of https://github.com/EnriqueNeyra/eInkFrame

Displays images on e-ink digital frame. Also creates a webserver at port 8080 where you can upload and manage images.

To install:
- Raspberry Pi running PiOS lite 64-bit
- Clone the repo
- create a service for both image_upload_server.py and image_viewer.py
- upload images and enjoy
